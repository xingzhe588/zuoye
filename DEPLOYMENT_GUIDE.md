## 💻 Системные требования

### Минимальные требования:
- **Node.js**: версия 16.0 или выше
- **npm**: версия 8.0 или выше (или yarn 1.22+)
- **Оперативная память**: минимум 4 ГБ
- **Свободное место**: минимум 2 ГБ

### Рекомендуемые требования:
- **Node.js**: версия 18.0 или выше
- **npm**: версия 9.0 или выше
- **Оперативная память**: 8 ГБ или больше
- **Свободное место**: 5 ГБ или больше

### Поддерживаемые операционные системы:
- Windows 10/11
- macOS 10.15 или выше
- Ubuntu 18.04 или выше
- Другие Linux дистрибутивы

## 🛠️ Установка и настройка

### 1. Клонирование репозитория

```bash
# Клонируйте репозиторий
git clone <repository-url>
cd artcollab-master

# Или если у вас уже есть архив
unzip artcollab-master.zip
cd artcollab-master
```

### 2. Установка зависимостей

```bash
# Установка всех зависимостей
npm install

# Или с использованием yarn
yarn install
```

### 3. Настройка переменных окружения

Создайте файл `.env` в корневой директории:

```bash
# .env
REACT_APP_API_URL=http://localhost:3001
REACT_APP_GIGACHAT_API=https://api.gigachat.example.com
REACT_APP_ENVIRONMENT=development
```

### 4. Проверка установки

```bash
# Запуск тестов для проверки
npm test

# Проверка сборки
npm run build
```

## 🔧 Запуск в режиме разработки



```bash

npm test -- --coverage
```

## 📦 Сборка для продакшена


## 🌐 Развертывание

### Развертывание на Netlify

1. **Подготовка:**
   ```bash
   npm run build
   ```

2. **Загрузка:**
   - Перейдите на [netlify.com](https://netlify.com)
   - Перетащите папку `build/` в область развертывания
   - Или подключите GitHub репозиторий

3. **Настройка:**
   - Build command: `npm run build`
   - Publish directory: `build`

### Развертывание на Vercel

1. **Установка CLI:**
   ```bash
   npm install -g vercel
   ```

2. **Развертывание:**
   ```bash
   vercel --prod
   ```

### Развертывание на собственном сервере

1. **Подготовка сервера:**
   ```bash
   # Установка nginx
   sudo apt update
   sudo apt install nginx

   # Создание директории для приложения
   sudo mkdir -p /var/www/artcollab
   ```

2. **Загрузка файлов:**
   ```bash
   # Копирование сборки на сервер
   scp -r build/* user@server:/var/www/artcollab/
   ```

3. **Настройка nginx:**
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;
       root /var/www/artcollab;
       index index.html;

       location / {
           try_files $uri $uri/ /index.html;
       }

       location /static/ {
           expires 1y;
           add_header Cache-Control "public, immutable";
       }
   }
   ```

## 👥 Руководство пользователя

### Для новых пользователей

1. **Первое посещение:**
   - Откройте приложение в браузере
   - Ознакомьтесь с главной страницей
   - Попробуйте создать искусство (2 бесплатные попытки)

2. **Регистрация:**
   - Нажмите "Вход/Регистрация" в навигации
   - Выберите вкладку "Регистрация"
   - Заполните форму и подтвердите email

3. **Создание искусства:**
   - Перейдите в раздел "Создать"
   - Введите описание для ИИ-генерации
   - Дождитесь результата и сохраните работу

### Основные функции

#### 🎨 Создание искусства
- **Доступ:** Требует авторизации (гости: 2 попытки)
- **Способы:** ИИ-генерация, загрузка файлов, шаблоны
- **Формат:** Поддержка PNG, JPG, GIF
- **Ограничения:** Максимум 10 МБ на файл

#### 👤 Личный кабинет
- **Профиль:** Редактирование личных данных
- **Коллекция:** Управление созданными работами
- **Настройки:** Персонализация интерфейса
- **История:** Просмотр активности

#### 🔍 Просмотр коллекций
- **Публичные работы:** Доступны всем пользователям
- **Фильтрация:** По категориям, авторам, дате
- **Поиск:** Полнотекстовый поиск по описаниям
- **Сортировка:** По популярности, дате, рейтингу

### Горячие клавиши

- `Ctrl + /` - Открыть справку
- `Ctrl + K` - Быстрый поиск
- `Ctrl + N` - Создать новую работу
- `Esc` - Закрыть модальные окна

## ⚙️ Управление функциями

### Включение/отключение функций

```javascript
// В консоли браузера или в коде
import { featureManager } from './src/shared/config/features';

// Отключить создание искусства
featureManager.disable('artGeneration');

// Включить пользовательский центр
featureManager.enable('userCenter');

// Проверить статус функции
featureManager.isEnabled('auth'); // true/false

// Получить все функции
featureManager.getAllFeatures();
```

### Доступные функции

| Функция | Описание | По умолчанию |
|---------|----------|--------------|
| `auth` | Система аутентификации | ✅ Включена |
| `userCenter` | Личный кабинет | ✅ Включена |
| `artGeneration` | Создание искусства | ✅ Включена |
| `collection` | Коллекция работ | ✅ Включена |
| `contact` | Страница контактов | ✅ Включена |
| `navigation` | Навигационное меню | ✅ Включена |

### Настройка функций для разных сред

```javascript
// config/features.js
const features = {
  development: {
    auth: true,
    userCenter: true,
    artGeneration: true,
    collection: true,
    contact: true,
    navigation: true
  },
  production: {
    auth: true,
    userCenter: true,
    artGeneration: true,
    collection: true,
    contact: false, // Отключено в продакшене
    navigation: true
  }
};
```

## 🔧 Устранение неполадок

### Частые проблемы

#### 1. Ошибка "Module not found"
```bash
# Очистка кэша и переустановка
rm -rf node_modules package-lock.json
npm install
```

#### 2. Проблемы с портом
```bash
# Изменение порта
PORT=3001 npm start

# Или в .env файле
echo "PORT=3001" >> .env
```

#### 3. Ошибки сборки
```bash
# Увеличение памяти для Node.js
export NODE_OPTIONS="--max-old-space-size=4096"
npm run build
```

#### 4. Проблемы с CORS
```javascript
// В package.json добавить proxy
"proxy": "http://localhost:3001"
```

### Логи и отладка

#### Включение подробных логов
```bash
# Режим отладки
DEBUG=* npm start

# Логи webpack
npm start -- --verbose
```

#### Анализ производительности
```bash
# Профилирование сборки
npm run build -- --profile

# Анализ размера bundle
npm run analyze
```
